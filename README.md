## 全栈 Todolist Node(server) React(client) MongoDB(database) Typescript

### 写在最前面

-   这篇文章适用者，node 初学者的前端同学，或者对 react 有兴趣的后端同学。文章难度为入门，即使不太了解这些语言的同学稍微查阅资料也能看懂。
-   提示：需要对 Typescript 有一定了解，server 和 client 端均使用 Typescript
-   默认已经安装好 `yarn`或者`npm`
